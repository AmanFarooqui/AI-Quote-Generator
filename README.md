# AI Quote Generator using LSTM

A text generation application built using TensorFlow and LSTM networks.

## Features

- Quote generation using LSTM
- Interactive user interface
- Trained on 3000+ motivational quotes
- Temperature-based sampling

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Streamlit

## Project Structure

- Model Training Notebook
- Trained LSTM Model
- Tokenizer
- User Interface

pip install -r requirements.txt
